# Termux Setup Commands

This repository contains a complete set of commands to set up a basic Termux environment with Python, Git, PHP, and essential Python libraries.

## Features

- Update and upgrade Termux packages
- Install Python, Git, PHP, Curl, Wget
- Install Python libraries: `requests`, `mechanize`, `bs4`
- Setup storage access
- Check installation and storage permissions
- Cleanup unnecessary cache

## Installation / Setup Instructions

1. Open Termux and copy the commands from `Commands.md`.
2. Run commands **step by step** or as a script if using a shell script.
3. Allow storage permission when prompted.

## Notes
Make sure to select Group rotate and official mirrors when prompted by termux-change-repo.
Follow the commands in order to avoid errors.
This setup does not upgrade pip separately, as Termux handles pip versions safely.
